QuickBuy DevOps Project

QuickBuy is an e-commerce web application developed using PHP, JavaScript, MySQL and Docker. The project allows users to browse products, place orders through a shopping cart or Buy Now feature, and manage customer information and orders.

Main Features

- User Registration
- User Login
- Product Management
- Shopping Cart (Add products to cart and place orders)
- Buy Now Function
- Order Processing
- Customer Management
- Order History

Technologies

- PHP
- JavaScript
- MySQL
- Docker
- Docker Compose
- phpMyAdmin
- Git & GitHub

Pages

- index.php : Home Page
- sign.php : User Registration
- login.php : User Login
- addcart.php : Shopping Cart Orders
- olderaddtocart.php : Orders placed through the shopping cart
- olderbuynow.php : Orders placed through Buy Now
- payment.php : Direct order processing without using the cart
- orderp.php : Order Processing and Validation
- addproduct.php : Product Management
- enterccol.html : Admin Dashboard
